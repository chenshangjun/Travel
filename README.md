
### 开发环境和技术栈
 - 操作系统：window 10 64位
 - 开发工具：Visual Studio Code
 - 前端：Vue + Vuex + Vue-router + axios + Stylus

### 运行项目
1. 下载项目
2. 安装依赖 `npm install`
3. 启动项目 `npm run dev`
4. 访问`http://localhost:8080`
